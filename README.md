# 🤖 ZONE AI — বাংলা টেলিগ্রাম AI বট

একটি সম্পূর্ণ বাংলা UI টেলিগ্রাম বট, যা কাস্টম API ব্যবহার করে ইউজারের প্রশ্নের উত্তর দেয়।

## ✨ ফিচার লিস্ট
- 🤖 AI চ্যাট (আপনার দেওয়া API দিয়ে)
- 📢 বাধ্যতামূলক চ্যানেল জয়েন (যত ইচ্ছা তত চ্যানেল যুক্ত করা যায়)
- 🔢 দৈনিক রিকোয়েস্ট লিমিট (ডিফল্ট ১০টি, এডমিন প্যানেল থেকে পরিবর্তনযোগ্য)
- 🛠 সম্পূর্ণ এডমিন প্যানেল (`/admin` কমান্ড):
  - API URL/Key পরিবর্তন
  - চ্যানেল যুক্ত/রিমুভ
  - লিমিট পরিবর্তন
  - কাস্টম বাটন যুক্ত/রিমুভ
  - প্রিমিয়াম কাস্টম ইমোজি সেট
  - স্ট্যাটাস দেখা (মোট ইউজার সংখ্যা ইত্যাদি)
  - সব ইউজারকে ব্রডকাস্ট মেসেজ পাঠানো
- 😎 প্রিমিয়াম ইমোজি সাপোর্ট (custom_emoji_id দিয়ে)

## ⚙️ ইনস্টলেশন

```bash
pip install -r requirements.txt
```

## 🔧 সেটআপ (bot.py ফাইলে)

`bot.py` ফাইলের শুরুতে ২টা জিনিস বদলাতে হবে:

```python
BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"   # @BotFather থেকে পাওয়া টোকেন
ADMIN_IDS = [123456789]            # আপনার টেলিগ্রাম ইউজার আইডি (নিউমেরিক)
```

- **BOT_TOKEN** পাবেন টেলিগ্রামে `@BotFather` কে মেসেজ করে `/newbot` কমান্ড দিয়ে।
- **ADMIN_IDS** (আপনার নিউমেরিক আইডি) পাবেন `@userinfobot` অথবা `@myidbot` কে মেসেজ দিয়ে। একাধিক এডমিন দিতে চাইলে: `ADMIN_IDS = [111, 222]`

## ▶️ বট রান করা

```bash
python bot.py
```

## 📢 চ্যানেল জয়েন ফিচার কীভাবে কাজ করে
1. `/admin` কমান্ড দিয়ে এডমিন প্যানেল খুলুন
2. "📢 চ্যানেল যুক্ত করুন" চাপুন
3. চ্যানেলের ইউজারনেম দিন (যেমন `@mychannel`)
4. চ্যানেলের একটা নাম দিন (ইউজারদের দেখানোর জন্য, যেমন `আমাদের অফিসিয়াল চ্যানেল`)
5. **⚠️ অবশ্যই বটটিকে সেই চ্যানেলে Admin বানিয়ে দিতে হবে**, নাহলে জয়েন-চেক কাজ করবে না।

## 🔗 API যেভাবে কাজ করে
ডিফল্টভাবে আপনার দেওয়া API সেট করা আছে:
```
https://r-bots-free-apis.co08.art/api/v1/api/deep-ai?query=...&apikey=...
```
এডমিন প্যানেল থেকে "🔗 API সেট করুন" চেপে যেকোনো সময় নতুন API URL/Key বসাতে পারবেন।

> 📝 **নোট:** প্রতিটি API-র রেসপন্স ফরম্যাট আলাদা হতে পারে (যেমন `{"response": "..."}` অথবা `{"answer": "..."}`)। বট স্বয়ংক্রিয়ভাবে `response`, `answer`, `result`, `message` — এই key গুলো চেক করে উত্তর বের করে। আপনার API-র রেসপন্সে ভিন্ন key থাকলে `database.py` ফাইলের `response_keys` লিস্টে সেটা যুক্ত করে দিন।

## 😎 প্রিমিয়াম ইমোজি সেট করা
1. `/admin` → "😎 প্রিমিয়াম ইমোজি সেট" চাপুন
2. প্রিমিয়াম ইমোজির `custom_emoji_id` পাঠান
   - এটা বের করতে: একটা প্রিমিয়াম ইমোজি কোনো "raw JSON দেখায়" এমন বটে ফরওয়ার্ড করুন, বা টেলিগ্রাম বট API-র `getUpdates` থেকে `custom_emoji_id` কপি করুন
3. না জানলে `skip` লিখুন — তখন সাধারণ ✨ ইমোজি দেখাবে

> প্রিমিয়াম কাস্টম ইমোজি শুধুমাত্র Telegram Premium থাকা ইউজারদের কাছে বিশেষ অ্যানিমেটেড ইমোজি হিসেবে দেখাবে; সাধারণ ইউজাররা ফলব্যাক ইমোজি দেখবে।
